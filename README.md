# Journal App using Ruby on Rails

This app is a journal app where a logged in user can create tasks and by different categories.


* Ruby version
```
$ ruby 3.1.2
$ Rails 7.0.3.1
```

* Setup
```
 $ bundle install
```

* Database creation
```
 $ rake db:setup
```

* How to run the test suite
```
$ rails test
```

* Starting the application
```
$ rails server
```
